# ShiftDeck 技术支持

更新日期：2026-03-27

## 适用范围

ShiftDeck 是一款面向轮班工作者的离线优先排班工具，用于管理轮班模板、手动调整班次、设置提醒、小组件展示以及 Apple Calendar 导出。

## 常见问题

- 如果首次进入 App 没有数据，请先在引导页选择轮班模板或创建自定义轮班周期。
- 如果提醒没有触发，请检查系统通知权限是否已授权。
- 如果 Apple Calendar 导出失败，请检查日历权限以及目标日历是否可写。
- 如果更换设备，请先在旧设备创建备份文件，再在新设备恢复。

## 排查建议

- 升级到最新版本后重试问题操作。
- 在“设置”页重新检查提醒、日历和备份配置。
- 如果是数据问题，优先导出备份后再进行恢复或重建。
