适用范围 / Overview

ShiftDeck 是一款面向轮班工作者的离线优先排班工具，用于管理轮班模板、手动调整班次、设置提醒、小组件展示以及 Apple Calendar 导出。

ShiftDeck is an offline-first shift scheduling app for shift workers. It helps manage recurring patterns, manual adjustments, reminders, widgets, backups, and Apple Calendar export.

常见问题 / FAQ

  • 如果首次进入 App 没有数据，请先在引导页选择轮班模板或创建自定义轮班周期。 If the app starts with no data, choose a template or create a custom cycle during onboarding.
  • 如果提醒没有触发，请检查系统通知权限是否已授权。 If reminders do not fire, check whether notification permission is enabled.
  • 如果 Apple Calendar 导出失败，请检查日历权限以及目标日历是否可写。 If Apple Calendar export fails, verify calendar permission and that the target calendar is writable.
  • 如果更换设备，请先在旧设备创建备份文件，再在新设备恢复。 If you move to a new device, create a backup on the old device first and then restore it on the new device.

排查建议 / Troubleshooting

  • 升级到最新版本后重试问题操作。 Update to the latest version and retry.
  • 在“设置”页重新检查提醒、日历和备份配置。 Recheck reminders, calendar access, and backup settings in the Settings page.
  • 如果是数据问题，优先导出备份后再进行恢复或重建。 For data issues, export a backup before restoring or rebuilding data.
